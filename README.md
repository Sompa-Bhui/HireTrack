# HireTrack

Production-ready MERN job application tracker with authentication, analytics, Kanban workflow, resume management, and export tools.

## Stack

- React 19 + Vite
- Tailwind CSS
- React Router
- React Hook Form
- Axios
- Node.js + Express.js
- MongoDB Atlas + Mongoose
- JWT authentication
- Cloudinary uploads
- Recharts
- React Hot Toast
- Framer Motion
- React Icons
- @dnd-kit

## Setup

1. Install dependencies in `client` and `server`.
2. Configure environment variables from the examples.
3. Run `npm run dev` from the repo root.

## Notes

This repo includes a production-oriented starter architecture with:

- MVC backend
- Protected routes
- Dashboard analytics
- Kanban board
- Resume upload/download flow
- Markdown notes
- Search, filters, pagination
- Theme support
